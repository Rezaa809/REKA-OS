## 🌌 Tentang Proyek
**REKA-OS** bukan sekadar website portofolio biasa. Ini adalah simulasi lingkungan terminal yang memungkinkan pengunjung berinteraksi dengan proyek, data pribadi, dan sistem melalui baris perintah (*command line*). Dibangun dengan fokus pada kecepatan, desain minimalis, dan fungsionalitas murni.

### ⚡ Fitur Utama
- **Real Terminal Experience**: Navigasi intuitif menggunakan perintah teks sesungguhnya.
- **Intelligent Tab Completion**: Sistem pelengkap otomatis (Tab) untuk mempermudah eksekusi perintah.
- **Dynamic Themes**: Mendukung pergantian tema secara instan (termasuk mode Dracula).
- **Matrix Background**: Latar belakang animasi Matrix yang memberikan kesan futuristik.
- **Wiki Integration**: Panel samping (Wiki) untuk membaca dokumentasi tanpa menutup terminal.
- **OLED Optimized**: Skema warna yang ramah terhadap baterai dan mata pada layar OLED.

## ⌨️ Daftar Perintah (Cheat Sheet)
Setelah masuk ke sistem, kamu bisa mencoba perintah-perintah berikut:

| Perintah | Fungsi |
| :--- | :--- |
| `help` | Menampilkan semua daftar perintah sistem yang tersedia. |
| `ls` | Melihat daftar file dan folder dalam direktori saat ini. |
| `cd [nama]` | Masuk atau berpindah antar direktori. |
| `theme [nama]` | Mengganti tampilan visual (contoh: `theme dracula`). |
| `clear` | Membersihkan riwayat teks pada layar terminal. |
| `whoami` | Menampilkan informasi tentang pengembang (REKA). |
| `exit` | Kembali ke layar awal/login. |
